# vfox-goreleaser

[![CI](https://github.com/jbox-web/vfox-goreleaser/workflows/CI/badge.svg)](https://github.com/jbox-web/vfox-goreleaser/actions)

A plugin for the [mise version manager](https://github.com/jdx/mise) to install [goreleaser](https://github.com/goreleaser/goreleaser)

## Install

```
mise plugin install goreleaser https://github.com/jbox-web/vfox-goreleaser.git
```
